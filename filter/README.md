# ASR 情报清洗与结构化抽取流水线 (ASR Intelligence Pipeline)

本项目是一个工业级、高容错的 ASR（语音识别）文本处理与情报提取流水线。旨在从海量且充满识别噪声的电话语音转写记录中，精准提取并判定各类高危行为模式（如电信网络诈骗、涉毒交易、暴恐洗脑等）。

## 架构升级概述 (V4.0)

本项目最新架构抛弃了传统的“正则堆砌”与“线性加分”模式，全面转向**“软语义向量” × “硬句法特征” 的十字共现矩阵打分引擎**，并引入了基于编辑距离（Edit Distance）的模糊匹配来替换长句正则，极大地提升了系统对 ASR 错别字和近义词的抗噪泛化能力。

---

## 核心特性

### 1. 全链路漏斗分流设计 (Funnel Routing)
* **Stage One (阶段一)**：快速低成本过滤。负责文本规范化（NFKC）、无意义标点压缩、多语言 ASR 音似词纠错去重。支持根据轻量级特征（语气词占比、业务实体探针）将通话极速标记为 `PASS`（走全链路）或 `LITE`（静音/机器人，仅记录），避免昂贵的计算资源浪费。

### 2. 双轨对话拓扑引擎 (Y-Track Topology)
根据双方通话的字数分布和轮次交替节奏，自动将对话分类为两条分析轨道：
* **非对称轨道 (Asymmetric)**：主导方疯狂输出（如强推销、诈骗指令下达）。自动绑定 `Agent`（输出方）与 `Target`（接收方）。
* **对称轨道 (Symmetric)**：双方高频次互动协商（如毒品交易议价、杀猪盘前期情感铺垫）。自动绑定 `Driver`（业务驱动方）与 `Follower`（顺从方）。

### 3. 多维意图雷达 (Intent Radar)
采用 `BAAI/bge-m3` 多语言向量模型进行高精度的软语义提取。支持 11 个以上的核心意图槽位：
* **诈骗/控制**：诈骗黑话、验证码/屏幕共享客体、伪装公检法实体。
* **交易/违禁**：涉毒隐语、上下游交易链条。
* **群发/舆情**：有组织批量广播行为、煽动暴力、邪教洗脑、国际局势简报。
* **豁免/日常**：受害者摆烂抵抗 (Dismissal)、日常闲聊外卖、普通推销。

### 4. 容错型 NLP 硬特征提取 (Fuzzy Syntax Extractor)
废弃死板的长句正则表达式，采用分角色句法提取 + 模糊字面相似度匹配（Fuzzy Matching）。
* 自动锁定**骗子（Agent）**的专属话术，防止受害者提及敏感词引发的“角色行为污染”。
* 使用基于编辑距离的模糊匹配精准识别：`祈使压制句`、`假设条件胁迫 (如果不...就...)`、`资产盘问` 和 `连贯步骤引导`。

### 5. 十字共现矩阵打分器 (Co-occurrence Matrix Scorer)
* **矩阵交叉**：当 Agent 说出了特定的**硬特征（如恐吓条件句）**，且上下文出现了特定的**软意图（如扣费/惩罚的 BGE 向量结果）**时，产生高危矩阵核爆分。支持（硬×软）、（软×软）等任意组合。
* **沉默受害者防抖**：具备防线崩溃特征判定。只有当受害者有效发声 >= 3 轮时，才会记录防线收缩加分，完美过滤掉“把手机扔在一边”的沉默接听场景。
* **白名单豁免**：正规的电商/银行客服机器人可凭借其高度标准化的意图矩阵自动享受白名单折扣，极大降低业务客诉。

## 模块说明

| 文件名 | 职责模块 |
| :--- | :--- |
| `main.py` | 流程总编排。包含 CSV 超大单元格拆解、漏斗统计和路由决策入口。 |
| `models.py` / `models_stage2.py` | 采用 Pydantic 构建的数据结构模型，确保管道全生命周期内数据的不可变性与安全性。 |
| `stage_one_filter.py` | 第一阶段过滤器。进行文本归一化、语种识别(fastText)、结巴/音似词容错纠正。 |
| `stage_two_pipeline.py` | 第二阶段双轨引擎。提取深层对话统计学特征、NLP 硬句法分析与模糊匹配特征（解耦全局特征与角色特征）。 |
| `intent_radar.py` | 基于 BGE-M3 的软语义向量锚点雷达，支持精细化的相似度阈值重写。 |
| `role_binder.py` | 根据意图特征与对话拓扑模型，为每个说话人绑定交互角色与顺从度评分。 |
| `topology_engine.py` | 进行 ASR 文本碎片的同源合并与拓扑形态拆解。 |
| `stage_three_scorer.py` | 最终的共现矩阵打分引擎。计算高价值情报得分并生成带有详尽证据链的 JSON 情报记录。 |

## 环境依赖
由于本项目采用容错降级设计，可选择性安装以下核心后端库。若未安装对应的外部库，系统会自动平滑降级（如 BGE 降级为 N-gram Hash，HanLP 降级为基础正则）。

```bash
pip install pydantic
pip install numpy scikit-learn
pip install jellyfish flashtext pypinyin
```

**可选的高级 NLP 支持：**
* 软意图向量依赖：`FlagEmbedding` (BGE-M3)
* 依存句法与 NER 依赖：`ltp` 或 `hanlp`
* 语种识别依赖：`fasttext` (可使用 fasttext-wheel)

## 快速启动

直接运行入口文件。若配置的 `data/asr_records.csv` 不存在，系统会自动使用内置的三组 Mock 对话（典型诈骗、杀猪盘情感铺垫、纯未接通）进行模拟推演演示。

```bash
python main.py
```
